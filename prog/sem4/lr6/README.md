### ChainMap

Название типа данных: ChainMap
Не стандартный тип данных и необходимо подключать из модуля ' collections'.

Методы и свойства:
- new_child(): Возвращает новый экземпляр класса ChainMap(), содержащий новый словарь m, за которым следуют все словари в текущем экземпляре.
- parents: Возвращает новый экземпляр класса ChainMap(), содержащий все словари в текущем экземпляре, кроме первого. Это полезно для пропуска первого словаря при поиске ключей.
- maps: Поддерживает все основные операции со списками, следовательно можно добавлять новые словари и удалять уже добавленные, а так же изменять их последовательность.
- get(key, default=None): Извлекает существующие ключи с помощью поиска ключей в стиле словаря. Если ключ не найден, возвращает default.

Особенности:
- Поиск осуществляется последовательно по сопоставлениям до тех пор, пока не будет найден ключ.
- Если одно из сопоставлений обновляется, эти изменения будут отражены в ChainMap.
- Группа нескольких словарей или других сопоставлений вместе для создания единого, обновляемого представления.


### UserDict

Название типа данных: UserDict
Не стандартный тип данных и необходимо подключать из модуля ' collections'.

Методы и свойства:
- items(): Возвращает пары ключ-значение.
- values(): Возрващает все значения.
- update(): Обновляет словарь значениями из другого словаря или итерируемого объекта.
- keys(): Возвращает все ключи.

Особенности:
- Работает как оболочка вокруг стандартного словаря, добавляя новые методы и атрибуты.
- Упрощает работу со словарями за счет предоставления более простого интерфейса.

### UserList

Название типа данных: UserList
Не стандартный тип данных и необходимо подключать из модуля ' collections'.

Методы и свойства:
- init(iterable) — инициализирует новый объект UserList с элементами из iterable;
- append(item) — добавляет элемент item в конец списка;
- extend(iterable) — расширяет список, добавляя элементы из iterable;
- insert(index, item) — вставляет элемент item в список по индексу index;
- remove(item) — удаляет первое вхождение элемента item из списка;
- pop(index=-1) — удаляет элемент по индексу index и возвращает его;
- index(value, start=0, end=len(list)) — возвращает индекс первого найденного значения value в списке или -1, если значение не найдено;
- count(value) — возвращает количество вхождений значения value в список;
- sort(key=None, reverse=False) — сортирует список на месте;
- reverse() — меняет порядок элементов списка на противоположный;
- copy() — создаёт копию списка;
- clear() — очищает список, удаляя все его элементы.

Особенности:
- Это пользовательский класс, который позволяет добавлять атрибуты к стандартным типам данных, таким как списки и словари.
- Можно использовать для добавления новых методов и свойств к существующим объектам списков и словарей.
- Помогает избежать конфликтов имён между новыми атрибутами и существующими методами и свойствами стандартных типов данных.

### UserString

Название типа данных: UserString
Не стандартный тип данных и необходимо подключать из модуля ' collections'.

Методы и свойства:
- upper(): Возвращает строку в верхнем регистре.
- lower(): Возвращает строку в нижнем регистре.
- capitalize(): Возвращает строку с заглавной буквы.

Особенности:
- Атрибут UserString.data предоставляет доступ к обернутому строковому объекту.
- Поддерживают методы и операции строк.
